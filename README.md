## 📋 About
MindEase is a web-based mental wellness portal designed for UiTM students to perform self-assessment screening and connect with counselors.

## 🛠️ Tech Stack
- **Framework:** CakePHP
- **Database:** phpMyAdmin (SQL)
- **CSS:** Bootstrap 5
- **Local Server:** Laragon

## ⚙️ Installation
1. Clone repository
2. Import `database/mindease.sql` into phpMyAdmin
3. Configure `config/app_local.php`
4. Run via Laragon
